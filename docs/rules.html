Semantic matching

Rules match based on semantic similarity. Embeddings handle discovery—no manual glob patterns required.

4 rule types

Pattern (coding standards), Constraint (validation), Semantic (context-aware guidance), Automation (code generation).

Usage tracking

Felix tracks which rules the AI applied, accepted, modified, or rejected. Effectiveness scores guide auto-degradation.

Auto-degradation

When linked code is deleted or rules go unused for 30+ days, Felix marks them inactive. Catalog stays fresh without manual cleanup.

Claude Code hooks

Felix injects applicable rules via user_prompt_submit hook. Rules appear in the conversation before the AI responds.

Conflict detection

Felix automatically detects when rules have overlapping or conflicting guidance, helping maintain a consistent rule catalog.
